Purpose:

This program is a simplified spreadsheet program. Its purpose is to track words, numbers, and formulas. When specified with an "=" in front, this will attempt to calculate as a formula. All sheets made in this program are saved with a .sprd, for example "NewSheet.sprd". Basic functionality include creating new, Save As/Saving, and opening a Spreadsheet file. Extra Features include taking a screenshot of the Spreadsheet & allowing the User to Show/Hide details along the Program.
