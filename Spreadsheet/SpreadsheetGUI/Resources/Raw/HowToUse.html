File:

New:

Opens a new (blank) spreadsheet. If there is Unsave changes, it will ask the User (you) want to save. "Yes" will ask you to name it, Saving the file will create a new file. Otherwise if it does exist, it updates the file. "No" will creates new spreadsheet without saving. "Cancel" will not creating a new Spreadsheet.

Open:

Opens a file and copies into a new spreadsheet. To open a file, it should end in ".sprd" for this Program. If the spreadsheet has unsave changes, it will ask you if you want to save. "Yes" will ask you to name it, Saving the file will create a new file. Otherwise if it already exist, it updates the pre-existing file. "No" will discard it changes. Otherwise, opens and copies it content of the file into the spreadsheet.

Save:

Saves the spreadsheet. If the spreadsheet already exist, quickly save the contents & update the file. Otherwise, same feature as Save As.

Save As:

Let the User (you) save the spreadsheet at a specific location. The spreadsheet will ask you for a path & name, it'll then create a new saved file.

View :

Screenshot:

Let the User be able to take a screenshot of the spreadsheet, Would ask the User to first name the snapshot, it will then create a .PNG file that will be save in the User Documents folder on any devices

Show:

Allow the User to hide more of the GUI feature in not showing the Current Cell & Value text boxes. More focus in entering the content if the User chooses.

Help:

About:

This provides information about the explanation purpose of the Program, updates, Versioning, and Author for the program.

How to Use:

This provides information on it's feature & how to use the Spreadsheet works.

GUI Properties:

Current Cell:

a non-editable text box showing the cell name of the selected cell.

Value:

a non-editable text box showing the cell value of the selected cell.

Enter Content:

an editable text box showing the contents of the selected cell. The User can type and edit information about the cell. When the contents begins with an '=' then the cell is calculated as a formula. The results are displayed in the value box.

How to add centent to a cell:

Click on "Enter Content" text box and enter what you want to add. After that, click on the "Enter Content" button or press "return" key (MacOS) or "Enter" key (Windows) to add it to the cell. If the content is a formula, make sure to have a "=" in front.

Spreadsheet/ Grid:

A table of cells that tracks the inputs from the user. Each cell has a name. The Grid consist of a window displaying a grid of 26 columns and 99 rows. A1 in the upper left and Z99 in the lower right. The value of each cell are displayed in the grid.

Formulas:

About:

Formulas represents a standard infix notation using standard precedence rules. The allowed symbols are non-negative numbers written using double-precision floating-point syntax (without unary preceeding '-' or '+'). Formula begins with the Equal sign "=" where the program will interpret the formula for calculations.

Allowed symbols:

The four types of allowed symbols in a formula are:

  1. Non-negative numbers written using double-precision floating-point syntax.
    0, 1, 10.01, 69.420, 69e10
  2. Variables are Cell names, which consists of one letters from A-Z & followed by one or more digits from 1-99. Cell names are case insensitive. For example, "A2" and "a2" are treated as the same.
    a1, A10, C9, M69, Z99
  3. Parentheses are symbols used in pairs to group things together or specify the order of operations in an equation.
    ( , )
  4. The four operator symbols
    In order: Addition, Subtraction, Multiplication, Division
    + , - , * , /
Limitations:

For this Program, a Formula are not capable of handling negative numbers or performing implied multiplication

=(2 - 3) * 4
